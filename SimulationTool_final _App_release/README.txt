Simulation Tool
***************

1. First, extract the folder and save all the contents of the folder together at one place, i.e In one folder

2. The contents of the folder include howtoenter.txt, Instructions.txt, operatorguide.txt, README.txt and SimulationTool.jar

3. To use the tool in Windows, just double click on the executable file, i.e SimulationTool.jar

4. To use the tool in Linux, enter the command "java -jar SimulationTool.jar" on the terminal

**Important: You must have java runtime environment installed on your systems to use the tool.
You can download the java runtime environment from http://www.oracle.com/technetwork/java/javase/downloads/index.html
